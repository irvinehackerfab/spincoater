# Full Spincoater Program

## Required Libraries (download in Library Manager in Arduino IDE)
- [ESP32Servo](https://github.com/madhephaestus/ESP32Servo)
- [GUIslice](https://github.com/ImpulseAdventure/GUIslice)
- [TFT_eSPI](https://github.com/Bodmer/TFT_eSPI)
- [XPT2046_Touchscreen](https://github.com/PaulStoffregen/XPT2046_Touchscreen)

## How to Compile:
You must modify some of the libraries after installing them. See [this issue](https://github.com/irvinehackerfab/spincoater/issues/6#issuecomment-3937901951) for instructions.
